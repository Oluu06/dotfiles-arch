## üé® First Steps

### 1. Add Wallpapers
```bash
# Copy your wallpapers
cp /path/to/wallpaper.jpg ~/.local/share/wallpapers/

# Set wallpaper
wallpaper-changer
```

### 2. Configure Prompt
```bash
# Open terminal and run
p10k configure
```

### 3. Explore Themes
```bash
# Try different themes
theme-switcher
```
