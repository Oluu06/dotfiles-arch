# Nord Theme

An arctic, north-bluish color palette.

## Colors

- **Background**: `#2e3440`
- **Foreground**: `#eceff4`
- **Primary**: `#88c0d0` (Cyan)
- **Secondary**: `#b48ead` (Purple)
- **Alert**: `#bf616a` (Red)

## Wallpapers

Add your wallpapers to the `wallpapers/` directory.

Default wallpaper should be named `default.jpg`.
