# Tokyo Night Theme

A clean, dark theme inspired by the lights of Tokyo at night.

## Colors

- **Background**: `#1a1b26`
- **Foreground**: `#c0caf5`
- **Primary**: `#7aa2f7` (Blue)
- **Secondary**: `#bb9af7` (Purple)
- **Alert**: `#f7768e` (Red)

## Wallpapers

Add your wallpapers to the `wallpapers/` directory.

Default wallpaper should be named `default.jpg`.
