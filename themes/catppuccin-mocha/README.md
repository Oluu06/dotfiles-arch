# Catppuccin Mocha Theme

Soothing pastel theme for the high-spirited!

## Colors

- **Background**: `#1e1e2e`
- **Foreground**: `#cdd6f4`
- **Primary**: `#89b4fa` (Blue)
- **Secondary**: `#f5c2e7` (Pink)
- **Alert**: `#f38ba8` (Red)

## Wallpapers

Add your wallpapers to the `wallpapers/` directory.

Default wallpaper should be named `default.jpg`.
